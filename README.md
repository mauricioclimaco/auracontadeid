🎓 Exercício Capgemini School - Aura Account ID

Projeto prático desenvolvido durante a Capgemini School para demonstrar a integração entre componentes Aura e o back-end Apex.
🚀 Funcionalidades

    Recuperação de registros de Conta (Account) via ID.

    Comunicação assíncrona entre Front-end e Servidor.

    Interface baseada em Lightning Design System (SLDS).

🛠️ Stack Técnica

    Apex: Controlador AuraEnabled com consultas SOQL.

    JavaScript: Manipulação de eventos e lógica de client-side (Controller/Helper).

    Aura Components: Estrutura de interface e atributos.

📁 Estrutura

    StartComponent.cmp (Marcação)

    StartComponentController.js (Eventos)

    StartComponentHelper.js (Chamadas de Servidor)

    StartComponentController.apxc (Back-end)
